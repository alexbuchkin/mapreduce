Большие тесты вырезаны из-за размера

Тесты лежат в папке small/input, соответствующие записи командных строк - в папке small/cmdline. Если скопировать содержимое файла из папки small/cmdline с любым номером в терминал, текущая директория которого "csv", и запустить, то программа создаст выходной файл с соответствующим номером и с разрешением .csv в папке small/output
